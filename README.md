# Demo Site Generator

Generate instant, professional demo websites for local service businesses. Built for sales calls — show prospects what their website could look like in seconds.

**[Live →](https://website-demos.opscorescale.com)**

## Features

- Generate a full demo site from a business name and service type
- Admin panel with PIN-protected site management
- Google review import and display
- Hero image selection from curated library
- Logo upload via Vercel Blob
- Mobile-responsive designs
- Multiple service templates (plumbing, electrical, handyman, etc.)

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Vercel Blob (image storage)
- Deployed on Vercel

## Run Locally

```bash
npm install
npm run dev
```

Requires a `.env.local` with Vercel Blob credentials.
